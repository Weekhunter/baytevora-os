# BDL Icons

## Icon Sizes

BOS defines a small set of icon size constants. No icon pack is implemented this sprint; these tokens ensure future icons are applied consistently.

| Token | Value | Usage |
|-------|-------|-------|
| Small | 16 px | Inline indicators, status badges, compact lists |
| Medium | 20 px | Small toolbar buttons, inline actions |
| Normal | 24 px | Standard toolbar and list icons |
| Large | 32 px | Launcher icons, prominent actions |
| ExtraLarge | 48 px | Empty states, splash graphics |

## Guidelines

- Use the smallest size that remains clearly legible at the target screen density.
- Reserve `Large` and `ExtraLarge` for single-icon moments.
- Pair icon sizes with the surrounding text scale (e.g., `Normal` icons next to `Body` text).
