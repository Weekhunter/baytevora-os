# BDL Spacing

## Spacing Scale

Spacing values are based on a 4 px grid. All margins, paddings, gaps, and offsets should reference these tokens.

| Token | Value |
|-------|-------|
| space4 | 4 px |
| space8 | 8 px |
| space12 | 12 px |
| space16 | 16 px |
| space20 | 20 px |
| space24 | 24 px |
| space32 | 32 px |
| space40 | 40 px |
| space48 | 48 px |
| space64 | 64 px |

## Guidelines

- Use `space8` and `space12` for compact internal padding.
- Use `space16` and `space24` for card and panel margins.
- Use `space32` and above for section or page-level separation.
- Pair related elements with `space8` or `space12`.
- Avoid arbitrary pixel values such as 13, 17, or 21 px.
